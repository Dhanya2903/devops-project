# DevOps Project

## Overview
This project demonstrates an end-to-end DevOps pipeline.

## Tools Used
- GitHub
- Jenkins
- Docker
- Kubernetes
- Terraform
- Prometheus & Grafana (Planned)

## Workflow
1. Developer pushes code to GitHub
2. Jenkins builds pipeline
3. Docker container is created
4. Kubernetes deploys application
5. Terraform provisions infrastructure
6. Monitoring using Prometheus & Grafana
